Chapter 01

Why veth Exists

Modern Linux networking centers on isolation. veth pairs act as virtual cables: bidirectional pipes inside the kernel. They connect isolated namespaces with near-zero latency.

veth pair
[ veth0 ] <═══════════════> [ veth1 ]
Key InsightThe entire veth pair lives in kernel memory — no hardware, no serialization. Packets move at pointer speed.
02

Mental Model

Think physical cable: two ends, each in its own namespace. All standard tools (ip, tcpdump, ping) work transparently.

03

Network Namespaces

Each namespace has its own routing table, firewall, interfaces, and loopback. Perfect isolation.
